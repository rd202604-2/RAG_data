| 1 | 华斌 | 质量部 | 来料检验、入库检验、委外入库检验 | 1.下推来料检验单时，注意因检验进行的破坏性验证导致的不良，需要填写进样本破坏数栏位，不能写入不良栏位  2. 在下推检验单时，需要注意务必逐一物料下推检验单，不能批量下推，成品检验单一样也是要逐一汇报单下推对应的检验单，不要批量下推，批量下推会造成后续操作的困扰。 | 1.已经完成收料通知单需要下推检验单据的物料，希望可以对质量检验进行通知，通知质检需要下推检验单  2.成品入库汇报后需要下推检验单，希望可以对质检进行通知，通知质检下推检验单，确保及时性  3.不良批次的和不良品的数据目前要求不在金蝶中体现，后续如何实现正常将不良品数据录入系统. |
| 2 | 丁高明 | 财务部 |  | 见表格下方附件 |  |
| 3 | 刘影 | 财务部 |  | **发票管理：**  1、审核发票时，注意上查数据，保持单据之间的关联性  2、信利康的发票审核完之后，要做生产调整单，调整两个应付单之间的差额  **出纳管理：**  1、收款单：注意收款日期、用途、单据类型  2、付款单：注意付款日期、用途、单据类型、费用项目、部门  3、付款退款单：可以由付款单下推生成，但必须付款单的上游数据之间相关联，没有原始数据，不能下推生成，只能手动新增  4、公司所属银行之间的往来做银行转账单，单据类型为标准转账，美元结汇时单据类型选择购汇  5、筛选银行日记账时注意币别的选择，单据编号可汇总  **销售管理：**  销售订单：筛选时，注意日期，销售订单金额要与CRM金额保持一致，有的CRM单价固定的，有特殊原因，可有差异 | 1、含运费的收款不能够选单，不能做大于销售订单金额的收款，与对应订单不能相关联  2、付款单存在许多行时，即使在备注栏都填写内容，在列表里查看时却只显示一行内容  3、同一家的付款申请单，如果单据类型不同，不能合并下推生成付款单，只能单独下推 |
| 4 | 毛琳 | 市场部 | 订单 | 1、ERP销售订单下单时间需与CRM下单时间保持在同一月份（一般是同时下单的，特殊情况需注意两者的下单时间）；  2、OEM产品只要是工厂自己打印的，除了LOGO条和刘海贴片需要新建料号，其他的不需要建，而且BOM中也不需要体现；  3、需要报关的订单制作发货通知单，其他制作出库申请单；  4、OEM产品的BOM可以在标准品BOM的基础上修改，删掉不需要的，添加上新增的料号；  5、BOM版本重点需要注意的是底标签、纸盒标签、说明书、LOGO，这些都是OEM变动的重点部分；  6、注意电源需要单独下，扩展板默认带电源；  7、新建客户时，需要注意客户国家、结算币别和客户分组；  8、注意销售订单的CRM号不要输入错误；  9、新建料号，需要注意分组，并且可以找一个相同的复制修改；  10、新品标准品的BOM选择需与计划确认； |  |
| 5 | 吕芳 | 海外销售部 | 订单 | 1、客户下单：目前客户下单产品都比较固定，只有初次下单和配置BOM的时候有些不确定，但是通过请教同事和Hellen基本都能完成，就是下单耗时长短的问题。目前这块还算可以胜任。  2、下单审核：由于客户订单的所有细节都是通过邮件事先沟通过，所以比较清楚，因此正常审核订单没有问题；如果有问题的话就是有时候审核人对ERP上订单细节的显示方式不太清楚，需要和下单人充分沟通和说明。  3、发货通知：这块和之前的老金蝶类似，只要核对好数量和单价，注意电源规格和搭配数量，基本没有问题。  4、调拨：调拨到上海港区的仓库这块的工作目前只有Sangoma一家在操作，只要熟记步骤，一般也没有问题的。 | 有时候操作上有点问题需要撤回步骤重新修改，但是这个刷新周期会很长，还有的时候明明当时执行人已经撤回或者修改了操作，但是系统上迟迟都还不能显示。遇到紧急问题，会耽误宝贵的时间 |
| 6 | 胡呈彦 | 海外销售部 | 订单 |  | 1、新建客户，需要Alice审核，比之前麻烦； 2、新建物料审核人较多（Hellen, Jack, Emily），能否缩减 |
| 7 | 刘佳奇 | 海外销售部 | 订单 | 标准品：核对数量，型号即可，一般没什么问题 OEM 产品：重点关注和反复核对定制化的东西是否正确 | / |
| 8 | 周红 | 海外销售部 | 订单 | 见表格下方附件 |  |
| 10 | 夏琴 | 国内销售部 | 订单 | 目前基本操作顺畅，新增物料部分操作较少，比较生疏； 需要细心，不然重复修改和审核的次数要多 | 借测调拨部分新老金蝶有交叉，客户借测归还物品有缺零部件， 入库问题，目前还没有很好的处理方式 |
