如果按Solution建项目库，就需要将研发项目库里的问题全部移到相应Solution库。

首先开启全局权限中的**“**批量更改**”**权限，在项目中授予“**移动问题**”权限；

然后在搜索问题里根据项目和史诗链接筛选出相关所有问题，点击右上角“工具”，选择所有问题 → 选择需要移动的全部问题 → 选择“**移动问题**” → 选择新的项目和问题类型 → 确认所有域值将被保留 → 确认修改正确。
