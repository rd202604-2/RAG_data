1.  目的：

对公司文件进行命名， 确保本公司文件命名的一致性、正确性、适用性，确保相关部门及时得到并使用有效的版本，防止不适当的文件被误用。

2.  文件类型

2.1    证书类文件

2.2    规范类文件

2.3    产品配套资料文件

2.4    产品硬件类文档

2.5    产品软件类文档

2.6    部门模板类文档

3.  证书类文件命名规则

3.1    证书类文件分类：工商类，荣誉证书，知识产权，产品认证

3.2    证书类文件命名规则：证书名称全称（例：营业执照）

         知识产权类证书：知识产权类别\_名称 （例：软著\_HD2000网络电话软件）

         产品认证类证书：文件夹命名：认证类别（认证型号）（例：CE(UC923 UC903)）

                                      文件命名：型号\_认证类别\_文件内容（例：UC46\_CE\_EMC\_Certificate）

                                      当一个系列证书有3个及3个以上的型号时，文件命名：认证类别\_文件内容（型号）（例：CE\_Certificate(UC902P,UC902,UC601,UC601P)）

 4.  规范类文件命名规则：

     南京汉隆科技有限公司“规范名称”

5.  产品配套资料命名规则：

5.1    产品配套资料文件命名规则：

品牌名\_产品型号\_文档类型

品牌名：根据实际品牌命名（后续文件更新命名需要与创建时一致）

产品型号：英文大写

文档类型：中文描述（英文字母大写）

连接符：使用“\_”（下划线）

5.2    产品配套资料的类别及命名范例对照表：

|  |  |  |  |
| --- | --- | --- | --- |
| 文件类型 | 命名范例 | 文件类型 | 命名范例 |
| 底标签 | HTEK\_UC923\_底标签 | 快速安装手册 | HTEK\_UC923\_快速安装手册  HTEK\_UC923\_快速安装手册（中文版） |
| 纸盒 标贴 | HTEK\_UC923\_纸盒标贴 | 纸盒 | HTEK\_UC923\_纸盒 |
| 外箱 | HTEK\_UC923\_外箱 | 外箱 标贴 | HTEK\_UC923\_外箱标贴 |
| LOGO丝印设计 | HTEK\_UC923\_LOGO丝印 | 按键丝印设计 | HTEK\_UC923\_按键丝印 |
| 质保卡 | HTEK\_UC923\_质保卡 | OEM表 | Sangoma\_S505\_OEM表 |

6.  硬件产品资料命名规则（待讨论）

7.  软件产品资料命名规则（待讨论）

8.  部门模板命名规则

      要求对内统一使用中文名，对外根据需要由申请人自行决定语种

       例：

中文范例：汉隆付款申请单

           英文范例：Htek Distribution Agreement（代理协议）
