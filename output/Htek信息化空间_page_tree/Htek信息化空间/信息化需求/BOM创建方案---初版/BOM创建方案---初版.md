**一．****虚拟件命名规则：**

X X – X X – X X X X X X – X X

   ①   ②            ③           ④

①：阶段

   命名规则：

             成品：90

            OEM BOM：OEM

            话柄：80

            组装：70

            PCBA：60

②：客户代码：

     命名规则：(客户代码对照表)

            HTEK：HL

            Sangoma：SG

            Crexendo：CX

③：型号代码：

     命名规则：按照目前产品型号命名规则

④：版本代码：

     命名规则：三位数代码，从01顺次更新，下次变更更新为02，03……

例如：HTEK UC924的各阶代码

   成品代码：90-HL-UC924-01

   话柄代码：80-HL-UC924-01

   组装后半成品代码：70-HL-UC924-01

   半成品PCBA代码：60-HL-UC924-01

   OEM BOM代码：OEM-HL-UC924-01

**二．BOM分阶组成规则：（以HTEK UC924为例）**

1. 成品代码：90-HL-UC924-01

    成品BOM组成：包装物料 + 半成品BOM + OEM BOM + 话柄BOM

 2. 半成品代码：70-HL-UC924-01

    半成品BOM组成：组装物料 + PCBA BOM

 3. PCBA代码：60-HL-UC924-01

     PCBA BOM组成：电子物料

 4. 话柄代码：80-HL-UC924-01

     话柄BOM组成：话柄物料

 5. OEM BOM代码：OEM-HL-UC924-01

     OEM BOM组成：OEM物料（底标签+亚克力+说明书+网线+曲线+礼盒+礼盒标签+支架+外箱标签等）
