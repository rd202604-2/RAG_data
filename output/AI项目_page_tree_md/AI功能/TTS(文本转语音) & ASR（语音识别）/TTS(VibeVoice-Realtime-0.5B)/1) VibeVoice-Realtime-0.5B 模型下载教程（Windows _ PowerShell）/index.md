# [VibeVoice-Realtime-0.5B 安装教程（Windows / PowerShell）](#vibevoice-realtime-05b-安装教程windows--powershell)

本文档记录在 `C:\Htek\ARSTTS` 中安装 `microsoft/VibeVoice-Realtime-0.5B` 的完整步骤、实际执行结果与排障方案。

## [1. 已验证环境](#1-已验证环境)

* Python: `3.14.3`
* pip: `26.0.1`（虚拟环境内升级后）
* 工作目录: `C:\Htek\ARSTTS`

## [2. 安装步骤（可复现）](#2-安装步骤可复现)

### [2.1 创建虚拟环境](#21-创建虚拟环境)

```
cd C:\Htek\ARSTTS
python -m venv .venv
```

### [2.2 安装下载工具](#22-安装下载工具)

```
.\.venv\Scripts\python -m pip install --upgrade pip setuptools wheel huggingface_hub
```

### [2.3 下载模型到本地目录](#23-下载模型到本地目录)

```
.\.venv\Scripts\hf.exe download microsoft/VibeVoice-Realtime-0.5B --local-dir .\models\VibeVoice-Realtime-0.5B
```

## [3. 本次执行结果（真实记录，已完成）](#3-本次执行结果真实记录已完成)

### [3.1 成功项](#31-成功项)

* 虚拟环境创建成功：`.venv`
* 依赖安装成功：`huggingface_hub`、`hf.exe` 等已安装在 `.venv\Scripts`
* 模型下载成功：`microsoft/VibeVoice-Realtime-0.5B`
* 本地模型目录：`C:\Htek\ARSTTS\models\VibeVoice-Realtime-0.5B`
* 关键权重文件：`model.safetensors`（`2,035,332,888` bytes）
* 当前模型目录总大小：`2,035,471,506` bytes（含配置与缓存文件）

### [3.2 下载命令与实际输出](#32-下载命令与实际输出)

执行命令：

```
.\.venv\Scripts\hf.exe download microsoft/VibeVoice-Realtime-0.5B --local-dir .\models\VibeVoice-Realtime-0.5B
```

命令输出关键结果：

* `Fetching 6 files: 100%`
* 返回目录：`C:\Htek\ARSTTS\models\VibeVoice-Realtime-0.5B`

### [3.3 目录验证命令](#33-目录验证命令)

```
Get-ChildItem .\models\VibeVoice-Realtime-0.5B
Get-ChildItem .\models\VibeVoice-Realtime-0.5B -Recurse | Measure-Object -Property Length -Sum
```

## [4. 重装/补全下载命令（可重复执行）](#4-重装补全下载命令可重复执行)

```
cd C:\Htek\ARSTTS
.\.venv\Scripts\hf.exe download microsoft/VibeVoice-Realtime-0.5B --local-dir .\models\VibeVoice-Realtime-0.5B
```

下载完成后验证：

```
Get-ChildItem .\models\VibeVoice-Realtime-0.5B
```

## [5. 可选方案（内网/代理环境）](#5-可选方案内网代理环境)

如果你在公司内网，请先配置代理后再执行下载：

```
$env:HTTPS_PROXY="http://127.0.0.1:7890"
$env:HTTP_PROXY="http://127.0.0.1:7890"
.\.venv\Scripts\hf.exe download microsoft/VibeVoice-Realtime-0.5B --local-dir .\models\VibeVoice-Realtime-0.5B
```

如果无法直连 Hugging Face，也可以在可联网机器下载后，将整个目录拷贝到：

* `C:\Htek\ARSTTS\models\VibeVoice-Realtime-0.5B`
